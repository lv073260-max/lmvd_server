# LMVD WiFi Server

Herramienta ligera para la transferencia inalámbrica de archivos entre dispositivos Android y Windows a través de la red WiFi local. Ideal para mover fotos, vídeos y documentos rápidamente sin depender de cables ni servicios en la nube.

## Funcionalidades
- **Servidor Local**: Crea un punto de acceso para recibir archivos desde otros dispositivos
- **Multiplataforma**: Versiones optimizadas para dispositivos móviles (Android) y ordenadores (Windows).
- **Control Total**: Gestiona las descargas y transferencias desde una interfaz sencilla.

## Cómo empezar

### En Android (Móvil/Tablet/TV)
1. Instala el APK de `LMVD WiFi Server`.
2. Pulsa en "Iniciar Servidor".
3. Usa la dirección IP que se muestra para conectar desde otros dispositivos.

### En Windows (PC)
1. Abre la versión PC del servidor.
2. Configura la carpeta donde quieres recibir los archivos.
3. ¡Listo para compartir!
